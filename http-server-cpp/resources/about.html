About This Server

Project Overview

This is a multi-threaded HTTP server implementation built from scratch using C++ and low-level socket programming. The server demonstrates advanced computer networking concepts including concurrent programming, HTTP protocol implementation, and network security.

Key Features

  • Multi-threading: Configurable thread pool for handling concurrent clients
  • File Serving: Support for HTML, binary files (images, text)
  • JSON Processing: POST request handling with file uploads
  • Connection Management: Keep-alive support with timeout handling
  • Comprehensive Logging: Detailed request/response logging with timestamps
  • Error Handling: Proper HTTP status codes and error responses

Architecture

The server uses a producer-consumer pattern where the main thread accepts connections and adds them to a queue, while worker threads process requests from the queue. This design ensures efficient resource utilization and proper handling of concurrent clients.
