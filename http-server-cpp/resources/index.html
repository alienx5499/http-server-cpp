Multi-threaded HTTP Server

Welcome to the C++ HTTP Server built from scratch using socket programming!

Features

  • Multi-threaded with configurable thread pool
  • GET requests for HTML files and binary downloads
  • POST requests for JSON data uploads
  • Path traversal protection and host validation
  • Keep-alive connection support
  • Comprehensive logging with timestamps
  • Binary file integrity verification
